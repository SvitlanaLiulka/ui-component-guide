# Bootstrap Icons

Install the `bootstrap-icons` package using `npm i bootstrap-icons`.

Then add this `import 'bootstrap-icons/font/bootstrap-icons.css'` to the root file of project.

Visit [docs](https://icons.getbootstrap.com/#install) for detailed information and [icon list](https://icons.getbootstrap.com/#icons) for detailed documentation.

## Icon Props

Icons can be used using `i` tag. See the examples.

| Name      | Type            | default | Description                                                                                                    |
| --------- | --------------- | ------- | -------------------------------------------------------------------------------------------------------------- |
| className | text            |         | accepts the class of expected icon. For full list of icons visit [docs](https://icons.getbootstrap.com/#icons) |
| style     | `CSSProperties` |         | list of css properties as object                                                                               |
